Offcanvas Navbar

This example uses Offcanvas component to show collapsed navbar. The navbar has navbar-expand-lg class to set breakpoint to 922px. Use navbar-expand{-sm|-md|-lg|-xl|-xxl} for responsive collapsing or remove navbar-expand-* class to show the navbar-toggler on every screen size instead of the expanded menu.
